## `single_factor_auth_flutter`

Add `single_factor_auth_flutter` as a dependency to your `pubspec.yaml`

```yaml
dependencies:
  single_factor_auth_flutter: ^1.0.0
```

or

```sh
flutter pub add single_factor_auth_flutter
```
